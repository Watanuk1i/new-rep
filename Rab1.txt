Большая игра №1 — «Правило меньшинства»
Ставки до начала игры
1. Победитель игры
Кто станет финальным победителем?
Варианты: список участников

Расчёт: выигрывает ставка на последнего оставшегося игрока.

2. Дойдёт до финала
Дойдёт ли [игрок] до финальных 3 участников?
Да / Нет

Расчёт: если игрок попал в топ-3 — Да.

3. Первый вылет
Кто вылетит в первом раунде?
Варианты: список участников

Расчёт: если вылетело несколько игроков, выигрывают ставки на любого из вылетевших.

4. Будет ли ничья в игре
Будет ли хотя бы один раунд с равным количеством голосов?
Да / Нет

Расчёт: если хотя бы один вопрос дал ничью — Да.

5. Игра закончится быстро
Игра закончится за 5 раундов или меньше?
Да / Нет

Расчёт: считать только полноценные раунды без отменённых ничьей.

Ставки перед каждым раундом
6. Большинство выберет “Да”
На текущий вопрос большинство ответит “Да”?
Да / Нет

Расчёт: если “Да” набрало больше голосов — Да.

7. Меньшинство будет меньше 30%
В этом раунде меньшинство составит меньше 30% игроков?
Да / Нет

Пример: осталось 10 игроков, меньшинство 2 человека = Да.

8. Вылетит больше половины игроков
В этом раунде вылетит больше половины оставшихся игроков?
Да / Нет

Расчёт: если большинство оказалось больше половины всех активных игроков.

9. Автор вопроса пройдёт дальше
Игрок, задавший вопрос, пройдёт в следующий раунд?
Да / Нет

Расчёт: если он оказался в меньшинстве.

10. Будет массовый вылет
В этом раунде вылетит 5 или больше игроков?
Да / Нет

Можно менять число под количество участников. 

Большая игра №2 — «Комната девяти патронов»
Ставки до начала игры
1. Стрелок закончит игру в плюсе
Стрелок закончит игру с прибылью?
Да / Нет

Расчёт: после всех раундов смотрится итог Стрелка.

2. Заряжающий закончит игру в плюсе
Заряжающий закончит игру с прибылью?
Да / Нет

Расчёт: по итогам игры.

3. Кто потеряет больше всех
Кто закончит игру с самым большим убытком?
Варианты: список участников

Расчёт: по чистому минусу.

4. Кто заработает больше всех
Кто заработает больше всех?
Варианты: список участников

Расчёт: по чистому плюсу.

5. Будет ли долг
Хотя бы один игрок уйдёт в долг после игры?
Да / Нет

Расчёт: если у кого-то не хватило денег и создался долг.

Ставки перед раундом
6. В этом раунде будет попадание по Сидящему
В этом раунде красный патрон попадёт по Сидящему?
Да / Нет

Расчёт: если хотя бы один сидящий получил красный патрон.

7. В этом раунде попадут только по манекенам
Все красные патроны уйдут в манекены?
Да / Нет

Расчёт: если ни один Сидящий не получил красный патрон.

8. Стрелок будет в плюсе в этом раунде
Стрелок завершит этот раунд в плюсе?
Да / Нет

Расчёт: доходы Стрелка минус потери за конкретный раунд.

9. Сидящий на месте [номер] будет в безопасности
Место [номер] не получит красный патрон?
Да / Нет

Можно сделать для каждого места отдельно.

10. Будет перестановка мест
Стрелок переставит двух Сидящих перед выстрелами?
Да / Нет

Расчёт: если была использована перестановка.

12. Цена места превысит ¥100 000
Хотя бы одно место будет куплено дороже ¥100 000?
Да / Нет

Расчёт: по итогам аукциона мест.

13. Самое дорогое место
Какое место станет самым дорогим?
Варианты: места 1–9

Расчёт: место с самой высокой ценой покупки.

14. Кто сядет на самое опасное место
Кто окажется на месте, куда попадёт первый красный патрон?
Варианты: список сидящих / манекен

Расчёт: по первому красному попаданию.
